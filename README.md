# Soğan Commerce Patches

Centralized Composer patches for Soğan Commerce projects. Provides a single versioned source for all upstream patches applied across the ecosystem.

## Usage

This package is automatically required by `soganbilisim/sogan-commerce-project`. To use it standalone:

```bash
composer require soganbilisim/patches:^1.0
```
